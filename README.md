# java-cli-bazel-mongo-crud-update

## Description
Creates a user with credentials
and lists collection `test`.

## Tech stack
- java
- bazel
  - mongo connector
- mongo

## Docker stack
- mongodb:latest
- ol.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
